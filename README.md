# Global Reserve Monitor

A data-driven dashboard built to analyze and visualize the shifts in global currency reserves from 1999 to 2021. This project tracks how different reserve currencies have performed over the last two decades.

## 🚀 Features

* **Data Visualization:** Interactive charts displaying reserve trends over time.
* **Trend Analysis:** Insight into how dominant currencies have fluctuated in the global economy.
* **Streamlit Dashboard:** A user-friendly web interface for filtering and exploring the data.

## 🛠 Tech Stack

* **Python**
* **Streamlit** (for the web interface)
* **Pandas** (for data manipulation)
* **Git/GitHub** (for version control)

## 📂 Project Structure

* `app.py`: The main script to run the Streamlit dashboard.
* `load_data.py`: Script to process and clean the raw currency reserve data.
* `requirements.txt`: List of dependencies required to run the project.
